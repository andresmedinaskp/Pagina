Portafolio

Gestor de permisos

Este aplicativo es una herramienta diseñada para gestionar y controlar los permisos de los usuarios en una base de datos Firebird. Su función principal es permitir a los administradores de bases de datos asignar y administrar quién puede acceder a qué datos y qué acciones pueden realizar dentro de la base de datos.

Consolidación y Unificación de Archivos JSON

Este proyecto ofrece una solución eficiente para consolidar múltiples archivos JSON en uno solo, respetando y manteniendo la estructura definida en los datos originales. Permite combinar de manera automática y ordenada archivos JSON provenientes de diferentes fuentes, sin perder la integridad de la información y asegurando que la estructura de los datos se preserve a lo largo del proceso. La herramienta es capaz de gestionar archivos con estructuras complejas y jerárquicas, unificando los datos sin generar conflictos ni duplicados. Además, ofrece opciones para personalizar el proceso de consolidación, permitiendo definir cómo deben combinarse los diferentes elementos de los archivos, todo de manera sencilla y eficiente.

MisakLingua App

MisakLingua es una aplicación educativa diseñada especialmente para niños de la comunidad Misak, con el objetivo de enseñarles su idioma natal de manera divertida y efectiva, al mismo tiempo que refuerza el aprendizaje del español. A través de una serie de juegos interactivos, lecciones visuales y actividades prácticas, los niños podrán aprender y practicar tanto el Misak como el español de forma paralela. La app está diseñada para respetar y promover la riqueza cultural y lingüística de la comunidad Misak, ofreciendo contenidos adaptados a su contexto, valores y tradiciones. Con MisakLingua, los niños podrán mejorar su comprensión y expresión en ambos idiomas, favoreciendo el desarrollo de habilidades lingüísticas, cognitivas y sociales en un entorno amigable y accesible.

biblioteca Web

El proyecto Biblioteca Web Misak es una plataforma digital innovadora creada con el propósito de preservar, gestionar y difundir el rico patrimonio cultural del pueblo Misak. Esta biblioteca centralizada se presenta como una herramienta esencial para la población Misak, proporcionando acceso a una amplia gama de contenidos relacionados con su historia, tradiciones, saberes ancestrales, y conocimiento contemporáneo. El sistema está diseñado para facilitar la gestión de contenido digital de manera eficiente, permitiendo a los administradores cargar, organizar y actualizar información cultural, literaria y educativa de forma sencilla. Además, la plataforma ofrece funcionalidades avanzadas para la generación de reportes y estadísticas.

Sistema de Inventario y Ventas

El Sistema Integral de Inventario y Ventas es una solución avanzada diseñada para gestionar de manera eficiente tanto el inventario de productos como el proceso de ventas dentro de una organización. Este proyecto permite a las empresas controlar su inventario en tiempo real, registrar transacciones de ventas, y generar reportes detallados que faciliten la toma de decisiones estratégicas y operativas. Gestión de Inventario: El sistema permite un control exhaustivo del inventario, incluyendo la entrada y salida de productos, el seguimiento de niveles de stock, la gestión de proveedores y la identificación de productos en tiempo real. Los usuarios pueden realizar ajustes rápidos, generar alertas automáticas cuando los niveles de inventario sean bajos o productos próximos a caducar, asegurando una gestión eficiente y sin errores.
